# DoE — Design of Experiments

A small, dependency-light Python library for **designing experiments and analysing their
results**. It covers the two halves of the DoE workflow and the design matrix that joins
them:

1. **Design generation** — turn a set of factors into a structured set of runs (factorial,
   fractional-factorial, and response-surface designs).
2. **Analysis** — fit a model to the measured responses and tell you what matters (effects,
   interactions, curvature, significance, and an optimum).

Everything is built and fitted in **coded units** (continuous factors mapped to `[-1, +1]`)
but entered and reported in the **natural units** you actually set at the bench — the library
owns the translation.

## Why DoE?

Changing one factor at a time (OFAT) is blind to interactions and wasteful of runs. A
designed experiment varies factors *together* in a deliberate geometry, so every run informs
every effect — and a response-surface design then locates the optimum rather than just the
direction of improvement. See [`docs/VIGNETTES.md`](docs/VIGNETTES.md) for a worked, narrative
tour of these ideas (the running example optimises a cell transfection).

## Features

- **Factors & coding** — `ContinuousFactor` / `CategoricalFactor`, with natural↔coded
  translation handled for you.
- **Factorial designs** — `full_factorial` and `fractional_factorial` (2-level, from
  generator strings like `"D=ABC"`) for screening many factors cheaply.
- **Response-surface designs** — `central_composite` (CCD, with `faced`/`rotatable`/
  `orthogonal` axial spacing) and `box_behnken` for fitting curvature and finding optima.
- **Replication & run order** — `Design.replicate` and `Design.randomize`, with center-point
  tracking that feeds the pure-error / lack-of-fit estimate.
- **OLS analysis** — `fit_ols` returns coefficients, *effects* (the −1→+1 swing, `2×`
  coefficient), standard errors, t/p-values, confidence intervals, and R².
- **ANOVA & model checking** — `anova_table`, `lack_of_fit`, and predictive metrics
  `press` / `predicted_r2` (Q²) / `adjusted_r2`.
- **Plotting** (optional `matplotlib` extra) — `main_effects_plot`, `pareto_plot`,
  `half_normal_plot`, `contour_plot` (+ headless `surface_grid`), `residuals_vs_fitted`,
  `normal_qq`.

## Installation

The project uses [`uv`](https://docs.astral.sh/uv/) and a `src/` layout. The core library
depends only on `numpy`, `scipy`, and `pandas`; plotting is an optional extra.

```bash
uv venv && uv pip install -e '.'             # core library
uv pip install -e '.[plotting]'              # + matplotlib for the plots
uv pip install -e '.[dev]'                   # + pytest / ruff / mypy / matplotlib
```

Requires Python ≥ 3.11.

## Quick start

Run a replicated 2×2 factorial and read off the main effects and interaction:

```python
import numpy as np
from doe import ContinuousFactor, full_factorial, fit_ols

dna   = ContinuousFactor("dna_ng",   low=100, high=500, units="ng/well")
lipid = ContinuousFactor("lipid_uL", low=0.5, high=2.5, units="uL/well")

design = full_factorial([dna, lipid], levels=2)   # the 4 corners
rep    = design.replicate(3, each=True)            # 3 wells per corner

# measured % GFP+ per well (replace with your plate data)
gfp = np.array([22, 20, 24,  31, 29, 33,  40, 38, 42,  58, 60, 62], dtype=float)

result = fit_ols(rep, gfp, model="linear")   # mains + 2-factor interaction
print(result.summary().round(2))
#                  coefficient  effect  std_error      t    p
# term
# Intercept              38.25   38.25       0.58  66.25  0.0
# dna_ng                 11.75   23.50       0.58  20.35  0.0
# lipid_uL                7.25   14.50       0.58  12.56  0.0
# dna_ng:lipid_uL         2.75    5.50       0.58   4.76  0.0
```

Locate an optimum with a central composite design and a quadratic model:

```python
from doe import central_composite

# "faced" keeps every run inside the stated low/high bounds (no extrapolation)
ccd = central_composite([dna, lipid], alpha="faced", center=4)
y   = np.array([30.3, 37.0, 46.8, 66.9, 38.0, 60.7,
                47.1, 60.7, 60.0, 59.1, 60.9, 60.8])     # % GFP+ per run
fit = fit_ols(ccd, y, model="quadratic")
print(f"R^2 = {fit.r_squared:.4f}")                      # R^2 = 0.9966

from doe.plotting import surface_grid           # needs the `plotting` extra
X, Y, Z = surface_grid(fit, "dna_ng", "lipid_uL", resolution=101)
i, j = np.unravel_index(np.argmax(Z), Z.shape)
print(f"optimum: {X[i, j]:.0f} ng DNA, {Y[i, j]:.2f} uL lipid -> {Z[i, j]:.1f}% GFP+")
# optimum: 448 ng DNA, 2.38 uL lipid -> 67.1% GFP+
```

## Status

Phase 1 (factorial designs, OLS analysis, effect/Pareto/half-normal plots) and Phase 2a
(response-surface designs, quadratic fitting, ANOVA + lack-of-fit, contour/diagnostic plots)
are implemented. `plackett_burman` is a planned stub, and categorical-factor model expansion
is deferred. Optimal designs, space-filling, and mixture designs are future phases — see
[`docs/PLAN.md`](docs/PLAN.md).
